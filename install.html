Akses Lebih Cepat & Mudah

Install aplikasi PKM Baniona di perangkat Anda. Tidak perlu buka browser setiap saat — langsung buka dari layar utama.

🌐 Buka Aplikasi di Browser

Atau ikuti panduan di bawah sesuai perangkat Anda

Panduan Install di Android (Chrome)
1
Buka link aplikasi di Google Chrome:
puskesmas-baniona.github.io
2
Tunggu beberapa detik — Chrome akan menampilkan banner "Tambahkan ke Layar Utama" di bagian bawah layar.
3
Ketuk "Instal" atau "Tambahkan" pada banner tersebut.
4
Jika banner tidak muncul: ketuk ikon ⋮ (tiga titik) di pojok kanan atas, lalu pilih "Tambahkan ke layar utama".
5
Ikon aplikasi PKM Baniona akan muncul di layar utama perangkat Anda. ✅

Selesai! Buka aplikasi langsung dari ikon di layar utama seperti aplikasi biasa — tanpa buka browser lagi.

Panduan Install di iPhone / iPad (Safari)
1
Buka link ini di Safari (bukan Chrome atau browser lain):
puskesmas-baniona.github.io
2
Ketuk ikon Bagikan (kotak dengan panah ke atas ↑) di bagian bawah layar Safari.
3
Gulir ke bawah pada menu yang muncul, lalu ketuk "Tambahkan ke Layar Utama" (Add to Home Screen).
4
Ketuk "Tambahkan" (Add) di pojok kanan atas.
5
Ikon aplikasi PKM Baniona akan muncul di layar utama iPhone/iPad Anda. ✅

Penting: Gunakan Safari, bukan Chrome/Firefox — hanya Safari yang mendukung fitur "Add to Home Screen" di iOS.

Panduan Install di PC / Laptop (Chrome atau Edge)
1
Buka link berikut di Google Chrome atau Microsoft Edge:
puskesmas-baniona.github.io
2
Cari ikon ⊕ atau ikon komputer kecil di bagian kanan address bar (bilah alamat).
3
Klik ikon tersebut, lalu pilih "Install" pada dialog yang muncul.
4
Aplikasi akan terbuka di jendela terpisah dan ditambahkan ke taskbar / Start Menu. ✅

Tips: Jika ikon install tidak muncul, coba reload halaman atau buka via Menu ⋮ → Simpan dan bagikan → Instal halaman sebagai aplikasi.
